Description

Les décodeurs MAG sont dotés d’un matériel ultra-puissant qui garantit de bonnes performances lors de l’exécution des applications. En supportant les contenus haute définition tels que l’UltraHD, les boîtiers MAG vous permettent de regarder ces contenus avec la plus haute qualité.

FAQ

Comment ça fonctionne ?

Brancher votre MAG. 
Référencez-vous à votre manuel d'utilisation.

Comment ajouter les chaînes à votre MAG ?

Si vous souhaitez ajouter une liste IPTV sur votre MAG, vous avez besoin du lien portal :
Voici comment obtenir un lien portail :
1- Contactez-nous lors de votre achat afin de nous envoyer votre adresse MAC. 
2- Un portal vous serra envoyer. 
3- Ajouter le lien à votre appareil. 
Il ne vous reste plus qu'à appuyer sur le bouton d'actualisation de votre MAG, une fois l'application actualisée, elle devrait charger toutes les chaînes de votre IPTV.
Si toutefois l'actualisation ne se fait pas, redémarrer votre appareil.
